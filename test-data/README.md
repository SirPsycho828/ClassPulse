# Test Data CSVs

Upload these via **+ New Analysis** > choose **CSV / Spreadsheet** upload mode.

## CSV Formats

- **Grade For Me** (default): `Student Name, Q1, Q2, Q3...` with an `ANSWER KEY` row. ClassPulse grades automatically and detects misconceptions.
- **Already Scored** (`*-scored.csv`): `Student Name, Score` only. Use when you've already graded the work.
